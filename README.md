# 🍕 La Bella Restaurant - Multi-page Premium Restaurant Website

A complete, multi-page responsive website for a luxury restaurant.

## 🌟 Key Features
* **Multi-page Navigation:** Seamless transitions between Home, About, Menu, and Contact pages.
* **Responsive Design:** Mobile-first approach for foodies on the go.
* **Interactive Menu:** Categorized food menu with hover effects.
* **Booking System:** Functional contact/reservation form UI.

## 🛠️ Tech Stack
* **HTML5:** Semantic structure for better SEO.
* **CSS3:** Custom styling with Flexbox and Grid.
* **JavaScript:** Smooth scrolling and navigation toggles.
